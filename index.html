Frequently Asked Questions

সাধারণ প্রশ্ন ও উত্তর — everything you need to know

Word Window কী এবং এটি কীভাবে কাজ করে?
Word Window হল একটি বাংলা-বান্ধব English vocabulary শেখার প্ল্যাটফর্ম। প্রতিটি Lesson-এ শব্দ, অর্থ, উচ্চারণ এবং synonyms দেওয়া আছে। Lesson বাটনে ক্লিক করুন, শব্দ লোড হবে এবং Details বাটনে ক্লিক করে বিস্তারিত দেখুন।
আমি কি বাংলায় শব্দ অনুসন্ধান করতে পারি?
বর্তমানে Search বার দিয়ে ইংরেজি শব্দ অনুসন্ধান করা যায়। একটি Lesson সিলেক্ট করার পরে সার্চ বারে টাইপ করুন এবং রিয়েল-টাইমে ফলাফল দেখুন।
Speak বাটন কীভাবে কাজ করে?
প্রতিটি word card-এর "Speak" বাটনে ক্লিক করলে browser-এর built-in Web Speech API ব্যবহার করে সঠিক আমেরিকান English উচ্চারণে শব্দটি বলা হয়। এটি কাজ করার জন্য আপনার browser-এ Speech Synthesis সাপোর্ট থাকতে হবে।
Dark Mode কীভাবে চালু করব?
উপরের Navbar-এ 🌙 আইকন বাটনে ক্লিক করলে Dark Mode চালু হবে এবং ☀️ বাটনে ক্লিক করলে আবার Light Mode-এ ফিরে আসবে।
কতগুলো Lesson আছে এবং নতুন শব্দ কি যোগ হয়?
Lessons এবং vocabulary data Programming Hero-এর public API থেকে লোড হয়। Lesson সংখ্যা API আপডেটের উপর নির্ভর করে। প্রতিটি Lesson-এ বিভিন্ন স্তরের শব্দ রয়েছে যা আপনার vocabulary বাড়াতে সাহায্য করবে।
